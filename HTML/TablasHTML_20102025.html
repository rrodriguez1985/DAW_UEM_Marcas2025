Tablas en HTML5

Cuando hablamos de tablas en HTML, nos estamos refiriendo a un conjunto de celdas
donde podemos alojar distintos contenidos (imágenes, vídeos, texto, tablas,etc.)
Así las características que presenta una tabla son las siguientes:
  • Definimos tabla con la etiqueta de apertura y cierre table
  • La tabla va a estar dividia en filas definidas con la etiqueta tr
  • Cada fila, estára dividida en celdas representadas por la etiqueta td
  • Las celdas pueden contener texto, imágenes, formularios, listas,otras tablas, etc.

Elementos de una tabla

  • table: sirve para definir una tabla.
  • caption: sirve para poder colocar un título a una tabla
  • align="top/bottom": es un atributo con el que podemos
    alinear en la parte superior o inferior el título de la tabla
  • thead: Define la cabecera de una tabla
  • thDefine una celda como si fuera la cabecera de una tabla
  • tbody: Define el cuerpo de una tabla
  • tfoot: Define el pie de la tabla
  • tr: Define una fila de una tabla
  • td: Define una celda de una fila

Atributos específicos para tablas: combinación de tablas

Para poder hacer combinación de tablas, utilizaremos unas propiedades
que son rowsapn y colspan para combinar celdas de
las mismas.
  • rowspan:indica el número de filas que ocupa la celda.
    Es decir, sirve para combinar filas.
  • colspan:indica el número de columnas que ocupa una celda.
    Es decir, sirve para combinar columnas.

Propiedades de las celdas de las tablas

  • Border:Establece el ancho del borde de la tabla exterior en px
  • cellspacing:especificar el espacio entre celdas
  • cellpadding:especifica el espacio entre el borde de una celda y su contenido
