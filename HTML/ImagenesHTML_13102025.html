Imagenes en HTML5

En la informática, podemos distinguir dos tipos de imagenes:
  • Imágenes de mapa de bits: están formados por puntos de colores.
    Por Ejemplo: imágenes de videojuegos de 8 bits (supermario, imagénes de pixels, etc.)
  • Imágenes vectoriales: son imágenes que están formadas por formas geométricas.
    Por ejemplo: imágenes de icónos, ilustraciones, etc.

Ventajas y Desventajas de Imágenes Vectoriales (SVG)

  • Ventaja: ocupan poco espacio y se pueden ampliar a cualquier escala sin que pierda la calidad de imagen
  • Desventaja: NO permiten representar imágenes sin formas definidas.Ejemplo: una foto de un retrato.

Ventajas y Desventajas de Imágenes de Mapa de Bits (bpm)

  • Ventaja: Sí permiten representar imágenes sin formar definidas. P.E. Fotos con el móvil
  • Desventaja: Pierden mucha calidad al ampliarlas y ocupan mucho espacio en el disco duro.

Formatos de Fotos / Imágenes

  • JPG o JPEG: Son adecuados para fotos.Comprensión con pérdidas y conservan el color original
  • GIF: Adecuados para dibujos. No tiene pérdidas y tienen derecho de autor.
  • PNG: Adecuados para el uso general. No tiene pérdidas. Las fotos se convierten a este tipo de formato para su uso, y no permiten animaciones. Tiene transparencias.
  • WEBP: Adecuado para el uso en páginas web, puesto que tiene las características buenas en cuanto calidad de las JPG/JPEG, y la comprensión de las PNG.
    Son las más usadas por lo general en Servidores y Aplicaciones Web para representar imágenes.

Insertar imágenes en HTML5

Para poder insertar una imagen en una página web, tendremos que utilizar la siguiente estructura:
  • img:La etiqueta img, es una monoetiqueta, que utilizaremos para poder insertar imágenes.
  • Atributo src: El atributo src (source), nos ayudará a localizar la imágen que queremos insertar.
    Ejemplo: src="img/foto..."
  • Link de la dirección: Es la ubicación de nuestra imágen con el formato (extensión) que tiene.
    Ejemplo: foto.jpg, foto.jpeg, foto.png, foto.svg...

Atributos asociados a las imágenes / fotos en HTML5

  • src: indica la dirección (URI) de acceso a la imagen
  • alt: sirve para indicar un breve texto en caso de que la imagen falle.
  • longdesc: indica si es una URI de una descripción de una foto o imágen en caso de tener asociado un mapa de imágenes, describir imágenes.
  • height y width: sirven para dar un tamaño de altura y anchura a la foto. Siempre en px.
  • usemap: sirve para asociar a la imágen un mapa de imágenes (como un marcador)
  • ismap: sirve para definir un mapa de imágenes
  • title: a forma de tip, muestra información en un cuadradito amarillo de la imagen cuando pasamos el ratón por encima.

Figure en HTML5

Permite asociar un texto con una imagen.
  • figure:es la etiqueta padre. Es decir, el resto de etiquetas dependerán o caerán de esta.
  • img:es una etiqueta para insertar imágenes.
  • figcaption: es yba etiqueta que contendrá el texto asociado a la imagen.

Atributos que podemos usar tanto en audio como en vídeo

  • src:definir la url del audio o video
  • preload:es usado para almacenar vídeos o audios de gran tamaño temporalmente (buffering).
    • none: no almancena nada temporalmente
    • auto: almacena datos multimedia temporalmente
    • medatada: almacena temporalmente sólo los metadados del archivo
  • mediagroup: para realizar una agrupación multimedia
  • autoplay:reproducir desde el inicio
  • loop:reproducir en bucle
  • muted: silenciar audio
  • controls: mostrar los controles del reproductor
  • width y height: tamaño de anchura y altura a los vídeos
  • poster: poner una imagen previa al vídeo.
