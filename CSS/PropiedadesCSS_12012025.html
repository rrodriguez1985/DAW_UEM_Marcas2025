Propiedades CSS: Colores

Para poder establecer colores en CSS, podemos hacerlo de la siguiente manera:
  • Color: podemos establecer el color del texto de nuestra web
  • background-color: nos permite establecer el color de fondo de nuestra web o elemento

Para poder poner un color enun elemento CSS, podemos hacerlo de las siguientes maneras:
  • Palabra clave: Eje: red, yellow, green, blue, coral, etc.
  • RGB (Red, Green, Blue): Eje:rgb(0,58,205)
  • RGB pero con capa alpha: La capa alpha, lo que establece es una tonalidad más clara o más oscura.
    Tiene valores entre 0 (más clara) y 1 (más oscura). Eje: rgba(25,40,90,0.5)
  • Hexadecimal: Valores que van del 0-9 y de la A-F. Eje: #45FA5D
  • Hexadecimal abreviada: Tomamos como valor prioritario el primer dígito.Eje:#FF5628 --> #F52

Unidades en CSS

Dentro de las unidades de medida que podemos usar en CSS, podemos destacar dos grupos:
  • Unidades Absolutas: son un tipo de medida fija (no cambian nunca). Son ideales para medios
    que no son variables, puesto que son poco flexibles a la hora de usarlos. P.E. pulgadas, cm, picas,
    mm, pt, etc.
  • Unidades Relativas: son un tipo de medida más común en CSS, puesto que dependen de otra
    resolución y tienen un curva de aprendizaje que aunque son más complejas, son ideales para
    diferentes tamaños y para hacer webs responsive. Eje: em (1em = 16px), pixel, %, rem, etc.
  • width: para establecer anchura de nuestro elemento
  • height: para establecer altura de nuestro elemento

Modelos de Cajas en CSS

Todos los elementos HTML, cuando creamos páginas web, están dentro de lo que se conocen como cajas o
contenedores de contenido.
Las cajas que vamos utilizar, son principalmente divs (cajas) con etiquetas semánticas como pueden ser:
header, nav, footer, aside, article,etc.
Hasta ahora, utilizabamos tablas para maquetas, pero que estas son poco flexibles a la hora de poderlas
ajustar a distintas resoluciones, y por lo tanto, a la hora de realizar maquetación web, por lo que
vamos a utilizar cajas (div) con una serie de propiedades que son las que comentaremos a continuación:
  • Margin: corresponde al margen existente entre el borde exterior de nuestra caja/elemento
    contenedor y nuestro navegador
  • Padding: corresponde al margen interior de nuestra caja, es decir, el espacio existente
    entre el contenido y el borde de nuestro elemento contenedor
  • Contenido: son las etiquetas que van a ir dentro de nuestra caja o elemento contenedor

Elementos en Línea y Elementos en Bloque

  • Elementos en Linea: Son aquellos que se van sucediendo a lo largo de la misma lína de
    nuestra web, mientras vayan cogiendo uno detras de otros y de izquierda a derecha. Van por lo tanto,
    siguiendo el flujo de la orientación.
  • Elementos en Bloque: son elementos que, independientemente de la anchura que tengan, se separan
    verticalmente de los Elementos posteriores y anteriores. Es como si provocaran un salto de Línea
    antes y después de un elemento.

Modelos de cajas en CSS: Zonas de un elemento

Dentro de una caja, podemos encontrarnos las siguientes zonas a tener en cuenta:
  • Top: hace referencia a la parte superior del elemento
  • Left: hace referencia a la parte izquierda del elemento
  • Right: hace referencia a la parte derecha del elemento
  • Bottom: hace referencia a la parte inferior del elemento
  • Center: que hace referencia a la posición central entre dos elementos
Estos elementos, podemos aplicarlos de la siguiente manera:
  • margin-top o padding-top
  • margin-left o padding-left
  • margin-right o padding-right
  • margin-bottom o padding-bottom
La aplicación de estos elementos se pueden establecer de la siguiente manera:
  • margin o padding con un solo valor: aplicaría a los 4 márgenes de los elementos
    PE: margin / padding: 40px;
  • margin o padding con dos valores: aplicaría el primer valor a top/bottom,
    y el segundo a right/left de los elementos
    PE: margin / padding: 40px 20px
  • margin o padding con tres valores: aplicaría el primer valor a top,
    y el segundo a right/left, y el tercero a bottom de los elementos
    PE: margin / padding: 40px 20px 5px
  • margin o padding con cuatro valores: aplicaría el primero a top, el
    segundo a right, tercero a bottom y cuarto a left.
    PE: margin / padding: 40px 20px 5px 2px;

IMPORTANTE: Cuando se establecer valores, siempre se tiene que tener en cuenta que
se sigue el sentido de las agujas del reloj. Siendo en este caso, el primer elemento
top, el segundo right, tercero bottom, y por último left.
Esto no quita, que podemos aplicar un valor a un único lado en concreto si se requiere.
Ejemplo: margin-top:20px, donde se aplicaría únicamente un margen de 20px en la parte superior.

Tipos de Bordes en CSS

CSS, nos permite utilizar diversos tipos de propiedades para utilizar en los bordes de
cajas, imágenes u otros elementos. Para ello, podemos utilizar la siguiente propiedad:
  • border-color: ser utiliza para poder aplicar un color a un borde
  • border-width: se utiliza para poder aplicar un groso al borde.
    thin | medium | thick
  • border-style: define el estilo del borde en CSS

Los tipos de border-style que podemos encontrarnos son los siguientes:
  • none: no aplica ningún tipo de estilo y coge el valor por defecto.
  • hidden: "oculta" los bordes
  • dotted: establece puntos en los bordes
  • dashed: establece rayas discontinuas
  • solid: borde sólido
  • double: doble línea de borde
  • groove: biselado con luz desde arriba
  • ridge: biselado con luz desde abajo
  • inset: establece un borde con profundidad hacia dentro
  • outset: establece un borde con profundidad hacia fuera
