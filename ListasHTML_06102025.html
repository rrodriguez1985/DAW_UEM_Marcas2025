Listas en HTML5

En HTML5, disponemos de tres tipos de listas que nos van a permitir organizar el texto para darle un formato más conciso y claro.
Los tipos de listas que podemos utilizar son los siguietes:
  • Listas Ordenadas (ol): Son listas ordenadas, y van numeradas en orden.
    Las siglas ol, vienen de Ordered List.
  • Lista Desordenada (ul): Son listas desordenadas, es decir, no van enumeradas.
    Las siglas ul, vienen de Unordered List
  • Listas de Definición (dl): Son listas de deifinición.
    Las siglas dl, vienen dadas por Definition List.

Dento de un elemento lista, sea del tipo que sea, ponemos insertar cualquier tipo de elemento:
texto, imágenes, vídeos, tablas, audios, etc.
Por lo tanto, se pueden combinar listas ordenadas con desordenadas y viceversa.

Listas Ordenadas (ol)

La estructura de una lista ordenada (ol), es la siguiente:
  • La lista está formada por la etiqueta de apertura y cierre ol
  • Internamente, dispondrá de una etiqueta li en la que pondremos los elementos a listar,
    es decir, el contenido de nuestra lista.
    Los elementos que pongamos en li, serán los que se muestren en nuestra web.
  • La etiqueta li, irá siempre tanto en listas ordenadas como desordenadas, porque son parte de su estructura.

Ejemplo de Lista Ordenada:
  1. Real Madrid
  2. FC Barcelona
  3. Villareal
  4. Betis Balompie
  5. Getafe

Listas Desordenadas (ul)

La estructura de una lista desordenada (ul), es la siguiente:
  • La lista está formada por la etiqueta de apertura y cierre ul
  • Internamente, dispondrá de una etiqueta li en la que pondremos los elementos a listar,
    es decir, el contenido de nuestra lista.
    Los elementos que pongamos en li, serán los que se muestren en nuestra web.
  • La etiqueta li, irá siempre tanto en listas ordenadas como desordenadas, porque son parte de su estructura.

Ejemplo de Lista Desordenada:
  • Java
  • C/C++
  • JavaScript
  • Python
  • PHP

Formateado de Listas Ordenadas y Desordenadas

HTML5, nos permite en cierta medida poder customizar las listas a través de una serie de atributos básicos como el atributo start y
el atributo type
Con estos atributos, podemos cambiar el orden del comienzo de las listas ordenadas, y eltipo de icono (podemos poner letras, números normales o romanos, etc.).
En el caso de listas desordenadas, podemos poner iconos circulares (circle), discos(disc) y cuadrados (scuare).

Ejemplo de Lista Ordenada Aplicando atributos type y start

  1. Podemos poner letras en mayúsculas (A)
  2. Podemos poner letras en minúsculas (a)
  3. Podemos poner cifras romanas en mayúsculas y minúsculas (i I, ii II, iii III, iv IV...n)
  4. Podemos poner números normales (1,2,3,4...n)
  5. Podemos indicar en qué posición queremos que comience (utilizamos el atributo start)

Ejemplo de Lista Desordenada Aplicando atributos type

  • Podemos utilizar el tipo disc (viñetas redondas rellenas)
  • Podemos utilizar el tipo circle (viñetas redondas huecas)
  • Podemos utilizar el tipo square (cuadrados)

Listas de Definición en HTML5 (dl)

Las listas de definición, tienen una estructura totalmente distinta a las anteriores.
Tienen en este caso la siguiente estructura interna:
  • La etiqueta dl: Description List. Delimita la lista completa.
  • La etiqueta dt: Description Term. Delimitar el término de la lista
  • La etiqueta dd: Definition Description- Delimita la definición del término.

Ejemplo de Lista de Definición

dl
Crea la caja que contiene los items de la lista.
dt
Es el item que contiene un término a definir.
dd
Es el item que contiene la definición.

Listas anidadas en HTML

En HTML, podemos anidar diferentes tipos de listas combinando de esta manera listas ordenadas con desordenas y viceversa.
Suelen emplearse listas anidadas de hasta un 3er nivel.
