Elementos de Selección ID y CLASS

Tanto en HTML como en CSS, podemos utilizar una serie de elementos que se conocen
como elementos de selección, que nos permiten establecer características únicas
y concretas a nuestros diseños.
Para ello, utilizaremos los elementos ID y CLASS, que nos permitirán
pode establecer características a nuestras etiquetas según lo necesitemos.

Definición de ID

Un ID en HTML y CSS, se define como un atributo denominado como elemento único.
Es decir, un elemento que al ser aplicado en un documento HTML y/o CSS, no
puede repetirse en dicho documento.
Para identificarlos en html, utilizaremos el id segudio del nombre que se le quiere asignar
y en css, el # con el nombre asignado.

Definición de CLASS

El valor del atributo class, a diferencia del valor del atributo id, puede ser utilizado en más
elementos de un mismo documento aunque estos sean distintos.
El beneficio que tiene esto, es que cuando tengamos que aplicar el mismo estilo
a diferentes elementos HTML de un mismo documento, nos permitirá reducir
líneas de código en nuestros documentos .css o en nuestros css internos.
Para identificarlos en html, utilizaremos el class seguido del nombre que se le quiere asignar
y en css, el . con el nombre asignado.
