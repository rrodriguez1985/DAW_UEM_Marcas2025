Propiedad de Profundidad de Niveles (z-index)

La propiedad z-index, es un elemento que podemos usar en HTML y CSS, para establecer
profundidades en nuestras páginas web, en concreto en un elemento sobre el resto.
En concreto, esta propiedad nos va a permitir situar un elemento encima de otro,
y suele ser utilizado principalmente sobre imágenes.
Su funcionamiento es el siguiente:
  • Para poder establecer una profundidad, sólo tendremos que indicar el
    nivel de profundidad de dicho elemento.
  • Por ejemplo, si el nivel que le damos es mayor de 0, el elemento se
    posicionará por encima del elemento principal (z-index=4)
  • Por ejemplo, si el nivle que le damos es menor que 0, el elemento se
    posicionará por detrás del elemento principal (z-index=-2)
  • Si no le damos valor, o el valor que le damos es igual a 0, el elemento
    quedará tal y como aparece por defecto en nuestra web.
  • Importante: Tener en cuenta, que la capa o nivel 0, es la capa desde la
    que partimos, por lo que tanto, es la que está por defecto.

Es muy importante indicar que los valores de z-index así como las propiedades asociadas
a ests son las siguientes: left, top, right y bottom.
NO FUNICINAN con elementos que está posicionados con static o posición estática.
Es decir, su pocisionamiento debe ser distinto a static, y por lo tanto pueden ser: relative,
absolute, fixed, etc.
