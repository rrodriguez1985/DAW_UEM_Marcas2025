Hipertextos o Hipervínculos en HTML5

Los Hipervínculos o Hipertextos, son elementos característicos de las páginas web, puesto que permiten que las páginas web se relacionen entre sí.
Un enlace sólo puede llevar a un destino, pero enlaces diferentes pueden llevar al mismo destino.
El destino de un enlace puede ser un directorio, un archivo o un elemento (texto, tabla, imágen, etc.) situado en la página web.

La estructura que presenta es la siguiente:
  • Tenemos una etiqueta que se llama anchor (anclaje), definida por la etiqueta a.
    Esta etiqueta tiene apertura y cierre.
  • El atributo href. que es donde va ir el enlace al que queremos direccionarnos. Siempre irá entre comillas.
  • Un texto que va entre la etiqueta de apertura y cierre de "a" que es la que actuará como elemento de enlace para nuestros vínculos.

Para poder abrir el hipervínculo de la página web que queremos visitar en otra pestaña, utilizaremos el atributo target = "_blank".
Importante, cuando tratemos con documentos de cualquier tipo de formato, deberemos tener en cuenta la ubicación de dónde estamos y haciando dónde vamos...
Es decir, tener en "mente" el siguiente esquema: si estoy a la misma altura del documento al que quiero vincular, si estoy por encima o por debajo...

Para poder poner un hipervínculo a un correo, el procedimiento es muy similar al de vincular una web.
Para ello, lo que hacemos es lo siguiente: utilizaremos el atributo mailto e indicaremos el correo al que queremos vincular el link.
